# Authentication as User

* Language: ruby (1.8.7, 1.9.3, 2.0.0, 2.1, jruby, rubinius) 
* Dependencies: simple_oauth

## Install & Build

Make sure you have bundler installed, run `bundle install`

## Execute

Run the script

    $ ./execute.rb
